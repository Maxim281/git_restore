git-restore

Команда "restore" помогает отключить или даже отменить незафиксированные локальные изменения.
С одной стороны, эту команду можно использовать для отмены эффектов git add и unstage изменений, которые вы ранее добавили в промежуточную область.
С другой стороны, команда восстановления также может использоваться для отмены локальных изменений в файле, тем самым восстанавливая его последнее зафиксированное состояние.
